# 🏗️ 로컬 오픈소스 AI 제품 고민방

> 로컬 오픈소스 AI를 활용한 제품을 만들기 위한 의사결정 프레임워크.
> 각 단계를 순서대로 검증하고, 통과하면 다음 단계로 진행한다.

---

## 의사결정 파이프라인

```
┌─────────────────────────────────────────────────────┐
│  1단계: 아이템 선정                                   │
│  "뭘 만들 것인가?"                                    │
│  → 시장성 + 구조적 접근성 + 1인 실현 가능성            │
├─────────────────────────────────────────────────────┤
│  2단계: 도구 성능 검증                                │
│  "기반 모델이 이 작업을 할 수 있는가?"                  │
│  → 후보 모델 벤치마크 + 실제 태스크 정확도 테스트       │
├─────────────────────────────────────────────────────┤
│  3단계: LoRA 파인튜닝 가치 판단                        │
│  "기본 모델로 부족하면, 튜닝할 만한가?"                 │
│  → 데이터 확보 가능성 + 품질 개선폭 + 비용 대비 효과    │
└─────────────────────────────────────────────────────┘
```

## 폴더 구조

```
product_lab/
├── README.md                    ← 이 파일
├── 1_item_selection/            ← 아이템 후보 검토
│   ├── evaluation_criteria.md   ← 평가 기준표
│   └── candidates/              ← 후보별 상세 분석
├── 2_tool_benchmark/            ← 모델/도구 성능 검증
│   ├── model_candidates.md      ← 후보 모델 목록 + 스펙
│   └── test_results/            ← 실제 테스트 결과
└── 3_lora_evaluation/           ← LoRA 가치 판단
    └── lora_decision_matrix.md  ← 파인튜닝 의사결정
```

- [ ] 3단계: LoRA 필요성 판단
- [ ] 최종 Go/No-Go 결정
