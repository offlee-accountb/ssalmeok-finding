# 3B 로컬 모델 능력 전수조사 → 돈 되는 것 역추적

> 접근법: "뭘 만들까?" ❌ → "3B가 뭘 할 수 있지?" → "그 중에 돈 되는 건?" ✅

---

## 1. 3B가 할 수 있는 모든 것 (능력 카탈로그)

### 📝 텍스트 분류/판단 (정확도 높음 ⭐⭐⭐⭐⭐)
| # | 능력 | 입력 → 출력 | 예시 |
|---|------|------------|------|
| T1 | 감정 분류 | 텍스트 → 긍정/부정/중립 | 리뷰 분석 |
| T2 | 카테고리 분류 | 텍스트 → 라벨 | 이메일→업무/광고/개인 |
| T3 | 스팸 판별 | 텍스트 → 스팸/정상 | 메시지 필터 |
| T4 | 언어 감지 | 텍스트 → 언어코드 | ko/en/ja/zh |
| T5 | 중요도 판단 | 텍스트 → 상/중/하 | 알림 필터링 |
| T6 | 의도 분류 | 텍스트 → 질문/요청/정보/불만 | 고객응대 라우팅 |
| T7 | 유해성 판별 | 텍스트 → 안전/위험 | 콘텐츠 필터 |
| T8 | 주제 분류 | 텍스트 → 토픽 | 뉴스/문서 태깅 |

### 🔍 정보 추출 (정확도 높음 ⭐⭐⭐⭐)
| # | 능력 | 입력 → 출력 | 예시 |
|---|------|------------|------|
| E1 | 개체명 추출 (NER) | 텍스트 → 인명/지명/조직/날짜 | 계약서에서 당사자 추출 |
| E2 | 날짜/시간 추출 | 텍스트 → datetime | "다음 주 화요일 3시" → 2026-02-24T15:00 |
| E3 | 연락처 추출 | 텍스트 → 전화/이메일/주소 | 명함 사진 OCR 후 정리 |
| E4 | 금액 추출 | 텍스트 → 숫자+통화 | 영수증에서 합계 |
| E5 | 키워드 추출 | 텍스트 → 핵심 단어 리스트 | 문서 태깅 |
| E6 | 표 데이터 추출 | 텍스트 표 → JSON/CSV | 복붙한 표 정리 |
| E7 | TODO 추출 | 회의록 → 할일 리스트 | "김팀장이 금요일까지 보고서" |
| E8 | 관계 추출 | 텍스트 → A-관계-B | "삼성이 하만을 인수" |

### ✏️ 텍스트 변환 (정확도 중간 ⭐⭐⭐)
| # | 능력 | 입력 → 출력 | 예시 |
|---|------|------------|------|
| C1 | 요약 | 긴 텍스트 → 짧은 텍스트 | 1페이지 문서 → 3줄 |
| C2 | 번역 (짧은 것) | 한→영, 영→한 | 단문 번역 |
| C3 | 포맷 변환 | 비구조 → 구조 | 장문 메모 → 마크다운 |
| C4 | 문체 변환 | 구어체 → 문어체 | 카톡 → 이메일 |
| C5 | 맞춤법 교정 | 틀린 텍스트 → 교정 | 오타 수정 |
| C6 | 제목 생성 | 본문 → 제목 | 파일명 제안 |

### 🤖 구조화된 출력 생성 (정확도 높음 ⭐⭐⭐⭐⭐)
| # | 능력 | 입력 → 출력 | 예시 |
|---|------|------------|------|
| S1 | JSON 생성 | 자유 텍스트 → JSON | 데이터 정규화 |
| S2 | 명령어 생성 | 의도 → CLI 명령 | "이 파일 압축해" → `tar -czf` |
| S3 | 정규식 생성 | 설명 → regex | "전화번호 패턴" → `010-\d{4}-\d{4}` |
| S4 | SQL 생성 | 질문 → SQL | "이번달 매출 합계" → `SELECT SUM(...)` |
| S5 | 파일 경로 생성 | 규칙+파일 → 경로 | 날짜별/카테고리별 정리 경로 |
| S6 | 이벤트 생성 | 텍스트 → .ics | 약속 → 캘린더 이벤트 |
| S7 | API 호출 생성 | 의도 → HTTP 요청 | "슬랙에 메시지 보내" → POST payload |

### 🔀 판단/라우팅 (정확도 높음 ⭐⭐⭐⭐⭐)
| # | 능력 | 입력 → 출력 | 예시 |
|---|------|------------|------|
| R1 | 이진 판단 | 입력 → yes/no | "이 파일 삭제해도 되나?" |
| R2 | 다중 경로 선택 | 입력 → A/B/C | "이 요청은 누구한테?" |
| R3 | 우선순위 정렬 | 리스트 → 순서 | 할일 우선순위 |
| R4 | 매칭 | A와 B → 유사도 | 중복 파일 감지 |
| R5 | 필터링 | 리스트 → 조건 충족만 | 조건에 맞는 항목만 |

### 🖼️ 비전 (3B 멀티모달 모델, 예: Qwen-VL-3B)
| # | 능력 | 입력 → 출력 | 예시 |
|---|------|------------|------|
| V1 | 이미지 분류 | 사진 → 카테고리 | 영수증/명함/문서/사진 |
| V2 | 텍스트 읽기 (OCR) | 이미지 → 텍스트 | 스크린샷에서 텍스트 추출 |
| V3 | UI 이해 | 스크린샷 → 요소 설명 | "로그인 버튼 위치" |
| V4 | 간단한 도표 해석 | 차트 이미지 → 수치 | 파이차트 비율 읽기 |

## 2. 💰 돈이 될 수 있는 조합 (역추적)

위 능력들을 조합해서, "사람들이 돈 내고 쓸 만한 것"을 역으로 추적.

---

### 💰💰💰 A급: 명확하게 돈 됨

#### 아이디어 A1: "스마트 다운로드 폴더" (파일 자동 정리)
- **사용 능력**: T2 + E4 + S5 + R1 + C6
- **작동**: 다운로드 폴더 감시 → 파일 내용 읽기 → 카테고리 판단 → 이름 변경 → 폴더 이동
- **타겟**: 바탕화면 파일 300개인 모든 직장인
- **가격**: ₩12,000 일회성 or ₩3,000/월
- **기술 난이도**: 낮음 (파일시스템 감시 + 모델 호출)
- **해자**: "내용을 보고 판단"하는 건 규칙 기반으론 불가

#### 아이디어 A2: "클립보드 AI" (복사하면 자동 정리)
- **사용 능력**: T2 + E1~E4 + S1 + C3
- **작동**: 뭔가 복사 → AI가 유형 판단 → 자동 정리/변환
- **예시**: 전화번호 복사 → 전화번호부에 추가 제안, 주소 복사 → 지도 열기 제안, 영어 복사 → 번역 제안
- **타겟**: Mac 사용자 (Raycast/Alfred 대체)
- **가격**: ₩5,000/월
- **기술 난이도**: 중간 (클립보드 API + 앱 연동)

#### 아이디어 A3: "영수증/문서 자동 정리"
- **사용 능력**: V1 + V2 + E3 + E4 + S1 + S5
- **작동**: 사진 찍기/스캔 → AI가 종류 판단 + 금액 추출 → 자동 분류 저장
- **타겟**: 프리랜서, 자영업자 (경비 처리)
- **가격**: ₩9,900/월
- **기술 난이도**: 중간 (카메라/스캔 + 비전 모델)
- **해자**: 로컬 = 재무 데이터 유출 걱정 제로

#### 아이디어 A4: "이메일 자동 분류 + 요약"
- **사용 능력**: T2 + T5 + C1 + E7 + R3
- **작동**: 이메일 수신 → 중요도 판단 → 카테고리 태깅 → 핵심 1줄 요약
- **타겟**: 하루 50통+ 받는 직장인
- **가격**: ₩7,000/월
- **기술 난이도**: 중간 (이메일 클라이언트 연동)

---

### 💰💰 B급: 가능하지만 시장 검증 필요

#### 아이디어 B1: "회의록 → 할일 자동 추출"
- **사용 능력**: E2 + E7 + S6 + R3
- **작동**: 회의록 텍스트 붙여넣기 → 누가/언제/뭘 해야 하는지 추출 → 캘린더/할일앱에 등록
- **가격**: ₩5,000/월

#### 아이디어 B2: "로컬 데이터 변환기"
- **사용 능력**: S1 + S4 + C3 + E6
- **작동**: CSV↔JSON, 표↔마크다운, 비정형→정형 변환
- **타겟**: 비개발자 사무직
- **가격**: ₩8,000 일회성

#### 아이디어 B3: "알림 필터/요약" (Mac 알림 센터)
- **사용 능력**: T2 + T5 + R1 + C1
- **작동**: 모든 앱 알림 → AI가 중요도 판단 → 중요한 것만 표시 + 나머진 묶어서 요약
- **가격**: ₩3,000/월

#### 아이디어 B4: "자동 파일명 번역기"
- **사용 능력**: C2 + C6 + S5
- **작동**: 영문/일문 파일명 → 한국어로 자동 번역 + 날짜 정리
- **타겟**: 해외 거래처 문서 많은 사무직
- **가격**: ₩5,000 일회성
