## 1. 개요 & 내 포지션

### AI 데이터 라벨링이란?
AI 모델(GPT, Gemini 등)이 더 좋은 답변을 하도록 **사람이 평가하고 피드백**을 주는 작업.  
RLHF(Reinforcement Learning from Human Feedback)가 핵심 기법.

### 내 스펙 매칭

| 내 스펙 | 매칭 포지션 | 시급 영향 |
|---------|-----------|----------|
| 한국어 모국어 | 한국어 AI 트레이너 → **희소가치 높음** | 기본 $25~31/hr |
| 영어 중급 (SDSU 연수) | 이중언어 태스크 가능 | +$5~10 프리미엄 |
| 소재공학 (연세대) + LG전자 생기원 | STEM 도메인 전문가 → **고단가** | $40~100+/hr |
| 공공기관 관리자 (예산/기획/사업관리) | 정책/행정 도메인 지식 | 니치 차별화 |
| 인공신경망 주식투자 분석 기획 경험 | AI/ML 관심사 어필 | 보너스 |

## 3. 실행 전략 (투 트랙)

```
Track A: 한국어 전문가 (먼저 진입)          Track B: STEM 전문가 (나중에 도전)
─────────────────────────                ─────────────────────────
Outlier Korean Expert ←── 먼저 지원       Outlier STEM Expert ←── 실적 쌓고 지원
DataAnnotation.tech   ←── 동시 가입       Upwork RLHF 프로젝트 ←── 포트폴리오 후
Remotasks             ←── 동시 가입       
                                          
시급: $25~31                              시급: $40~100+
진입 난이도: 낮음                          진입 난이도: 중~높음
성공 확률: 80~90%                          성공 확률: 40~50% (실적 쌓으면 올라감)
```

**전략 핵심:**
- Track A로 진입 → 리뷰 점수 축적 → 고단가 프로젝트 배정
- 여러 플랫폼 동시 가입 (한 곳에서 태스크 없으면 다른 곳에서)
- 리뷰어 승격 노리기 (단가 추가 상승)

## 4. 스크리닝 테스트 구조

### Outlier AI — 한국어 트랙

| 단계 | 언어 | 내용 | 난이도 |
|------|------|------|--------|
| Step 1. 가이드라인 숙지 | 🇺🇸 영어 | 평가 기준 읽고 이해. 시간제한 없음 | ★★☆ |
| Step 2. AI 응답 비교 | 🇰🇷 한국어 | 한국어 답변 2개 비교 → 더 나은 것 선택 | ★☆☆ |
| Step 3. 이유 설명 | 🇺🇸 영어 | 왜 그걸 골랐는지 2~3문장 영어 설명 ← **유일한 영어 작문** | ★★☆ |
| Step 4. 리라이팅 | 🇰🇷 한국어 | AI 응답을 더 자연스럽게 고쳐쓰기 | ★☆☆ |

### DataAnnotation.tech

| 단계 | 내용 | 소요시간 |
|------|------|---------|
| Starter Test | 간단한 퀴즈 | ~15분 |
| Core Test | AI 응답 평가 (good/ok/bad) + 에러 설명 | 1~1.5시간 |

### 테스트 핵심 팁
- 지시사항을 **극도로 꼼꼼히** 읽을 것
- AI 도구 사용 금지 (복붙 탐지됨)
- 집에서 혼자 응시
- 결과 통보: 빠르면 수시간, 늦으면 수일~수주

## 5. 영어 피드백 치트시트

### 🔄 비교/선택 표현

```
"Response A is better because..."
"Response B is more accurate/natural/comprehensive."
"While both responses address the topic, Response A is more [형용사]."
"Response A correctly uses [무엇], while Response B does not."
"I chose Response A for the following reasons:"
```

### ❌ 에러 지적 표현

```
"Grammatical error: [설명]"
"Factual error in [위치]: [설명]"
"Incorrect honorific: used [잘못된 것] instead of [올바른 것]"
"Unnatural phrasing: [원문] → should be [수정]"
"Missing context about [주제]"
"Inconsistent tone throughout the response"
"Contains hallucinated/fabricated information about [주제]"
```

### ✅ 긍정 평가 표현

```
"Accurately addresses the question"
"Uses appropriate tone and register"
"Well-structured and easy to follow"
"Factually correct and comprehensive"
"Natural-sounding Korean with proper grammar"
```

### 📝 개선 제안 표현

```
"Could be improved by adding [무엇]"
"Would benefit from more specific examples"
"Should use a more formal/casual tone for this context"
"The explanation of [주제] needs more detail"
"A more concise response would be preferable"
```

### 🏷️ 자주 쓰는 평가 용어

| 영어 | 의미 | 사용 맥락 |
|------|------|-----------|
| accurate | 정확한 | 사실관계 |
| fluent / natural | 자연스러운 | 한국어 표현 |
| awkward / unnatural | 어색한 | 번역투 표현 |
| factually correct | 사실적으로 정확한 | 팩트 체크 |
| culturally appropriate | 문화적으로 적절한 | 경어법, 호칭 |
| coherent | 논리적으로 일관된 | 글 흐름 |
| concise | 간결한 | 불필요한 반복 없음 |
| comprehensive | 포괄적인 | 빠진 내용 없음 |
| verbose / wordy | 장황한 | 너무 길게 씀 |
| relevant | 관련성 있는 | 질문에 맞는 답변인지 |
| off-topic | 주제에서 벗어난 | 엉뚱한 답변 |
| honorifics | 경어/존칭 | 한국어 특화 |
| register | 격식/비격식 수준 | 상황에 맞는 말투 |
| hallucination | 환각 (AI가 없는 사실 생성) | AI 평가 특화 용어 |
| well-structured | 구조가 잘 잡힌 | 글 구성 |
| repetitive | 반복적인 | 같은 말 반복 |

### 💬 실전 피드백 예문 (그대로 쓸 수 있는 수준)

```
예문 1 (간단):
"Response A is better. Response B uses incorrect honorific 
forms in a formal context."

예문 2 (중간):
"Response A is more accurate and natural. Response B contains 
a factual error about the manufacturing process described in 
paragraph 2. Additionally, Response B uses awkward phrasing 
that sounds like a direct translation from English."

예문 3 (상세):
"I selected Response A as the better response for three reasons:
1. It correctly uses formal honorifics appropriate for the context.
2. The explanation of the technical concept is factually accurate.
3. The overall tone is natural and culturally appropriate for 
   a Korean audience. Response B, while partially correct, 
   contains a hallucination about the date of the policy change."
```
