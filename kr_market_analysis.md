# 1. AutoApplier KR — AI 입사지원 자동화

## 1.1 원본 (미국) 분석

### 원본 제품: AutoApplier (autoapplier.com)

| 항목 | 내용 |
|------|------|
| 핵심 | AI가 채용공고 탐색 → 지원서 작성 → 제출 → 면접 코칭까지 자동화 |
| 런칭 | 2026.02 BetaList |
| 가격 | $30/월 (기본) ~ $90/월 (프로) |
| 수익모델 | 구독 SaaS |

### 핵심 기능 5가지

```
1. AI Job Agent
   └── 클라우드 브라우저가 실제로 채용사이트 열어서
   └── 지원서 작성 → 스크리닝 질문 응답 → 제출
   └── Workday, Greenhouse 등 ATS 시스템 대응
   └── 하루 수백 건 지원 가능

2. AI Interview Buddy
   └── 폰 마이크로 면접 대화 실시간 청취
   └── 이력서+JD 기반 맞춤 답변 제안
   └── GPT/Claude 기반 LLM 사용
   └── "탐지 불가" 주장 (폰에서 구동)

3. AI LinkedIn Bot (Chrome 확장)
   └── LinkedIn Easy Apply 자동화
   └── 블랙리스트 기업 설정
   └── 대시보드로 지원 현황 추적

4. AI Cover Letter Generator
   └── JD + 이력서 입력 → 맞춤 커버레터
   └── ATS 키워드 최적화

5. AI Resume Generator
   └── ATS 통과율 높이는 이력서 자동 생성
   └── 스마트 키워드 삽입
```

### 미국 경쟁사 현황

| 경쟁사 | 가격 | 차별점 |
|--------|------|--------|
| LazyApply | $99 일회성 or 구독 | LinkedIn/Indeed/ZipRecruiter 자동 지원 |
| JobCopilot | $8.90/주 (~$56/월) | 하루 50건 자동 지원, 콜백률 2% 미만 |
| Scale.jobs | $199 일회성 | AI + 사람 결합, 콜백률 47% 주장 |
| Sonara | 구독형 | 프로필 기반 자동 탐색+지원 |
| AIApply | $29/월 + 크레딧 | 크레딧 기반 지원 건당 과금 |

## 1.2 한국 시장 현황 (2026년 2월 기준)

### 이미 있는 것들

| 서비스 | 제공사 | 뭘 하나 | 한계 |
|--------|--------|---------|------|
| **AI 자소서 코치** | 사람인 | 자소서 항목별 AI 초안 생성 | 자소서 "생성"만. 지원 자동화 없음 |
| **LOOP AI 추천** | 잡코리아(워크스피어) | AI가 맞춤 공고 추천, 스와이프 지원 | 추천만. 자소서 자동 작성 안 함 |
| **뤼튼 / ChatGPT** | 범용 AI | 자소서 초안 작성 보조 | 범용 도구. 채용 특화 아님 |
| **무하유 PRISM** | 무하유 | AI 자소서 탐지 + 면접 평가 | ⚠ **기업용** (채용자 도구). 구직자 아님 |
| **그리팅** | 두들린 | 채용 관리 솔루션 (HR용) | ⚠ **기업용**. 구직자용 아님 |

### 👀 핵심 발견: 한국에 "없는 것"

```
미국 AutoApplier의 5가지 기능 중 한국에 있는 것:

✅ 자소서 AI 생성     → 사람인, 뤼튼 등이 부분적으로 제공
❌ 자동 지원 제출     → 한국에 없음 ← ★ 핵심 갭
❌ AI 면접 실시간 코칭 → 한국에 없음 ← ★ 핵심 갭
❌ 지원 현황 통합 대시보드 → 한국에 없음
⚠ 이력서 ATS 최적화  → 한국은 NCS 기반이라 다른 접근 필요
```

> **결론: "자소서 쓰기"는 이미 레드오션. "자동 제출 + 면접 코칭"이 블루오션.**

## 1.3 한국판 설계

### 타겟 고객

| 세그먼트 | 규모 | 페인포인트 | 지불의사 |
|----------|------|-----------|---------|
| **대졸 신입 구직자** | ~50만명/년 | 50곳+ 지원, 자소서 노가다 | 월 2~5만원 |
| **이직 준비생** | ~30만명/년 | 현직 유지하면서 몰래 지원 | 월 3~5만원 |
| **경력직 전환자** | ~20만명/년 | 업종 전환, 이력서 재구성 어려움 | 월 5만원+ |
| **공시생 전환자** | ~10만명/년 | 민간 채용 프로세스에 익숙하지 않음 | 월 2~3만원 |

### 기능 설계 (한국 특화)

```
AutoApplier KR v1.0
═══════════════════

[Core 1] 🔍 공고 통합 수집 + 매칭
├── 사람인, 잡코리아, 원티드, 캐치, 인크루트 크롤링
├── 관심 직군/지역/연봉/기업규모 필터
├── AI 매칭 점수: "이 공고, 너한테 87% 맞아"
└── 마감일 알림

[Core 2] ✍️ 자소서 AI 생성 (한국 특화)
├── 기업별 인재상 자동 분석 → 반영
├── 항목별 생성: 성장과정 / 지원동기 / 입사 후 포부 / 장단점
├── NCS 역량 키워드 자동 삽입
├── "이 기업은 '도전정신' 키워드 빈도 높음" 같은 힌트
├── 기업별 합격 자소서 패턴 학습 (크롤링 가능 범위)
└── 글자수 자동 맞춤 (500자/1000자/1500자)

[Core 3] 🚀 자동 제출 (1차 MVP에서는 제외 가능)
├── 사람인/잡코리아 자동 로그인 → 폼 입력 → 제출
├── 주의: 사이트별 봇 차단 정책 확인 필요
└── 1차에서는 "복붙 가이드" 수준으로 시작 가능

[Core 4] 🎤 AI 면접 코치
├── 인성면접 시뮬레이션: "자기소개 해주세요" → 피드백
├── 직무면접 시뮬레이션: 직군별 예상 질문 + 모범답안
├── 답변 분석: 길이, 구조, 키워드, STAR 기법 적용 여부
├── 음성 분석: 말 속도, 어, 음 빈도 (STT 기반)
└── 실시간 코칭 (고급): 실제 면접 중 폰에서 가이드

[Core 5] 📊 지원 대시보드
├── 어디에 언제 지원했는지 한눈에
├── 서류 통과 / 면접 일정 / 결과 추적
├── "이번 주 15곳 지원 → 서류 통과 3곳" 통계
└── 캘린더 연동 (면접 일정)
```

### 수익 모델

| 플랜 | 가격 | 내용 |
|------|------|------|
| **무료** | 0원 | 공고 탐색 무제한 + 자소서 3개/월 + 면접 연습 1회 |
| **기본** | 14,900원/월 | 자소서 무제한 + 면접 코치 5회/월 + 대시보드 |
| **프리미엄** | 29,900원/월 | 전 기능 무제한 + 자동 제출 + 음성 분석 |

| **기업 파트너** | B2B | 대학 취업센터 / 직업훈련원 단체 도입 |

### 수익 시뮬레이션

```
보수적 추정 (런칭 1년차):
├── 유료 사용자: 5,000명
├── ARPU: 20,000원/월 (기본+프리미엄 혼합)
├── 월 매출: 1억원
├── 연 매출: 12억원

└── 순이익률: ~60%+ (초기 마케팅 제외)

### 기술 스택 (MVP)

```
Frontend: Next.js + React
Backend: Node.js or Python FastAPI
DB: PostgreSQL + Redis
AI: OpenAI API (GPT-4o-mini for 자소서, Whisper for STT)
크롤링: Playwright / Puppeteer (채용공고 수집)
결제: TossPayments or 아임포트
배포: Vercel + AWS
```

### 리스크

| 리스크 | 수준 | 대응 |
|--------|------|------|
| 채용사이트가 자동 지원 차단 | 높음 | 1차에서는 "복붙 가이드"로 시작, 자동 제출은 나중에 |
| 기업이 AI 자소서 탐지 (무하유 PRISM) | 중 | 완전 AI 생성이 아닌 "인간 편집 보조" 포지셔닝 |
| 경쟁사 진입 (사람인 자체 개발 등) | 높음 | 선점 효과 + 면접 코칭 차별화 |
| 윤리 논란 | 중 | "AI 보조 도구"로 마케팅, 공정한 기회 강조 |

# 2. Nuanso KR — AI 광고 전략 브리핑

## 2.1 원본 (미국) 분석

### 원본 제품: Nuanso

| 항목 | 내용 |
|------|------|
| 핵심 | 광고 계정 연결 → AI가 성과 분석 + "다음에 뭘 해야 하나" 브리핑 |
| 런칭 | 2026.02 BetaList (매우 초기) |
| 타겟 | 소규모 사업자, 1인 마케터 |
| 차별점 | 대시보드(숫자 나열)가 아님 → "액션 제안"(뭘 하라고 알려줌) |

## 2.2 한국 시장 현황 (2026년 2월 기준)

### 이미 있는 것들

| 서비스 | 제공사 | 뭘 하나 | 한계 |
|--------|--------|---------|------|
| **카카오모먼트 AI** | 카카오 | 캠페인 진단, 최적화 점수, 업종 비교 | 카카오 광고 한정. 네이버/메타 통합 안 됨 |
| **ADVoost 쇼핑캠페인** | 네이버 | 예산만 넣으면 AI가 소재~최적화 자동 | 네이버 검색/디스플레이 한정 |
| **쉬운광고** | 카카오 | 모바일로 간단 광고 생성 | 제작 도구. 분석/전략 브리핑 없음 |
| **애드부스트 스크린** | 네이버 | AI 기반 옥외광고 자동 제작 | 옥외광고 특화 |

### 👀 핵심 발견: 한국에 "없는 것"

```
현재 한국 광고 AI 상황:

✅ 네이버 AI 광고 최적화  → 있지만 네이버 한정
✅ 카카오 AI 캠페인 진단   → 있지만 카카오 한정
✅ 메타(페이스북) AI 광고   → 있지만 메타 한정
❌ 전 채널 통합 AI 브리핑   → 없음 ← ★ 핵심 갭
❌ "뭘 해야 하는지" 액션 제안 → 각 플랫폼이 부분적. 통합 없음
❌ 비전문가용 쉬운 브리핑    → 대시보드는 있지만 해석이 어려움
```

> **결론: 각 플랫폼 안에 AI가 생기고 있지만, "전 채널 통합 + 한글 액션 브리핑"은 빈자리.**

## 2.3 한국판 설계

### 타겟 고객

| 세그먼트 | 규모 | 페인포인트 | 지불의사 |
|----------|------|-----------|---------|
| **스마트스토어 셀러** | 50만명+ | 네이버 광고 돌리는데 수치 해석 못 함 | 월 3~5만원 |
| **소상공인 (오프라인)** | 350만명 | 카카오/네이버 광고 하고 싶은데 어려움 | 월 2~3만원 |
| **1인 D2C 브랜드** | 5만명+ | 네이버+메타+카카오 멀티채널 운영 | 월 5~10만원 |
| **소규모 마케팅 대행사** | 1만곳+ | 클라이언트 여러 개 관리 필요 | 월 10~30만원 |

### 기능 설계

```
Nuanso KR v1.0 — "광고 브리핑 AI"
═══════════════════════════════════

[Core 1] 📊 광고 계정 통합 연결
├── 네이버 검색광고 API
├── 네이버 성과형 디스플레이 광고 API
├── 카카오모먼트 API
├── 메타(Facebook/Instagram) Marketing API
├── 구글 Ads API
└── (선택) 쿠팡 광고, 당근마켓 광고

[Core 2] 🤖 AI 주간 브리핑 (핵심 차별점)
├── 매주 월요일 아침 자동 발송
├── 한글 + 구어체로 작성 (전문 용어 최소화)
├── 예시:
│   "지난주 네이버 검색광고 요약:
│    '봄 원피스' 키워드 클릭단가가 23% 올랐어요.
│    경쟁 상점이 1주일 사이 4곳 늘었기 때문이에요.
│    → 추천: '봄 린넨 원피스'로 롱테일 전환하세요.
│    예상 절감: 주당 약 15,000원"
│
├── 채널별 성과 비교:
│   "네이버에서 ROAS 320% vs 메타에서 ROAS 180%
│    → 이번 주는 네이버 예산을 5만원 올리고
│       메타는 소재를 바꿔보세요"
│
└── 경쟁사 동향 (가능 범위):
    "같은 업종 평균 클릭률 2.3% → 내 클릭률 1.8%
     → 광고 소재 교체 추천"

[Core 3] 💡 원클릭 액션 제안
├── "예산 재배분하기" → 버튼 누르면 자동 적용
├── "이 키워드 중단하기" → 확인 후 적용
├── "새 키워드 추가하기" → 제안 목록에서 선택
└── (고급) AI가 소재 이미지/카피 초안 생성

[Core 4] 📱 알림 채널
├── 카카오톡 알림톡 (한국 최적)
├── 슬랙 연동 (소규모 팀)
├── 이메일 주간 리포트
└── 앱 푸시 (모바일 앱 버전)

[Core 5] 📈 리포트 자동 생성
├── 월간 광고 성과 리포트 PDF 자동 생성
├── 클라이언트에게 보내는 보고서 (대행사용)
└── 전년 동기 대비 / 전월 대비 시각화
```

├── LLM API 비용: ~월 1,000만원
├── 광고 API 연동 유지: ~월 300만원
└── 순이익률: ~50%+
```

### 기술 스택 (MVP)

```
Frontend: Next.js + React + Chart.js (시각화)
Backend: Python FastAPI (데이터 분석 특화)
DB: PostgreSQL + TimescaleDB (시계열 데이터)
AI: GPT-4o-mini (브리핑 생성) + 자체 룰 엔진 (액션 제안)
API 연동:
  ├── 네이버 검색광고 API (공식)
  ├── 카카오모먼트 API (공식)
  ├── Meta Marketing API (공식)
  └── Google Ads API (공식)
알림: 카카오 알림톡 API (비즈니스)
결제: TossPayments
배포: AWS
```

### 기술적 실현 가능성

| API | 공개 여부 | 접근 난이도 | 비고 |
|-----|----------|------------|------|
| 네이버 검색광고 API | ✅ 공개 | 중 | 인증 후 사용 가능 |
| 카카오모먼트 API | ✅ 공개 | 중 | 비즈니스 계정 필요 |
| Meta Marketing API | ✅ 공개 | 중~높 | 앱 심사 필요 |
| Google Ads API | ✅ 공개 | 높 | 개발자 토큰 신청+승인 |
| 쿠팡 광고 API | ⚠ 제한적 | 높 | 파트너사 한정 |

> 핵심 4개 채널(네이버, 카카오, 메타, 구글) API는 모두 공개.
> MVP는 네이버 검색광고 1개 채널만으로 시작 가능.

### 리스크

| 리스크 | 수준 | 대응 |
|--------|------|------|

| 광고 예산 0원인 소상공인 | 높음 | 월 광고비 10만원+ 쓰는 사업자만 타겟 |

| **선점 가치** | ★★★★★ (자동제출이 차별점) | ★★★★ (통합이 차별점) |

## 실행 시나리오

### 시나리오 1: AutoApplier KR 먼저

```
Month 1-2: 자소서 AI 생성 (NCS/인재상 특화) + 공고 수집
Month 3:   면접 코치 (텍스트 기반)
Month 4:   대시보드 + 유료화
Month 5-6: 면접 음성 분석 추가
Month 7+:  자동 제출 (법적 검토 후)
```

**장점:** 공채 시즌(3월, 9월)에 맞추면 폭발적 유저 유입 가능
**단점:** 시즈널 편차 큼

### 시나리오 2: Nuanso KR 먼저

```
Month 1-2: 네이버 검색광고 API 연동 + AI 브리핑
Month 3:   카카오모먼트 추가
Month 4:   유료화 + 카카오톡 알림
Month 5-6: 메타/구글 추가
Month 7+:  에이전시 플랜 + 리포트 자동화
```

**장점:** 비시즈널 (연중 꾸준한 수요)
**단점:** API 연동 초기 세팅에 시간 필요
