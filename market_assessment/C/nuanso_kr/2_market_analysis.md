## 2. 한국 경쟁 현황

### 기존 플레이어

| 서비스 | 제공사 | 뭘 하나 | 약점 |
|--------|--------|---------|------|
| **카카오모먼트 AI** | 카카오 | 캠페인 진단, 최적화 점수, 업종 비교 | 카카오 광고 한정 |
| **ADVoost 쇼핑캠페인** | 네이버 | 예산만 넣으면 AI가 소재~최적화 자동 | 네이버 한정 |
| **쉬운광고** | 카카오 | 모바일로 간단 광고 생성 | 제작 도구. 분석 없음 |
| **애드부스트 스크린** | 네이버 | AI 기반 옥외광고 자동 제작 | 옥외광고 특화 |

### 한국에 있는 것 vs 없는 것

```
✅ 이미 있는 것:
├── 네이버 자체 AI 광고 최적화 (ADVoost)
├── 카카오 자체 AI 캠페인 진단 (카카오모먼트AI)
├── 메타/구글 자체 AI 최적화
└── 각 플랫폼별 대시보드

❌ 한국에 없는 것 (Nuanso KR의 기회):
├── 전 채널 통합 AI 브리핑
├── "뭘 해야 하나" 한글 액션 제안 (비전문가용)
└── 멀티 채널 성과 비교 + 예산 재분배 추천
```

## 3. 🔴 핵심 장벽: 광고 데이터 접근

### 채널별 API 접근 현실

| 채널 | API | 성과 데이터 | 제3자 접근 조건 | 난이도 |
|------|:---:|:---:|------|:---:|
| **네이버 검색광고** | ✅ | 노출/클릭/비용/키워드별 | 광고주가 API키 발급+권한 부여 | 🟢 낮음 |
| **카카오모먼트** | ✅ | 캠페인/광고그룹/소재별 | **공식 대행사 or 카카오 사전 협의** | 🔴 높음 |
| **Meta (FB/Insta)** | ✅ | 풀 데이터 | 앱 심사 통과 | 🟡 중간 |
| **Google Ads** | ✅ | 풀 데이터 | 개발자 토큰 승인 | 🟡 중간 |

### ⚠️ 카카오모먼트 API 상세

```
접근 가능 대상 (2026년 기준):
  1. 카카오 공식 대행사 (KPP 파트너)
  2. 카카오와 사전 협의된 광고주

일반 스타트업이 접근하려면:
  → 카카오에 직접 문의 → 비즈앱 설정 → 협의 → 승인
  → 승인 기준이 명확하지 않음
  → 초기 스타트업이 통과하기 어려울 가능성 높음

제공 데이터 (접근 시):
  ✅ 광고계정/캠페인/광고그룹/소재 성과 지표
  ✅ 보고서 조회
  ✅ 광고 ON/OFF, 예산/입찰가 변경
  ✅ 일부 소재 생성/수정/삭제
```

### 네이버 검색광고 API 상세

```
접근 조건:
  ✅ 네이버 광고센터에서 API 사용 신청
  ✅ Customer ID + 액세스 라이선스 + 시크릿 키 발급
  ✅ 광고주가 제3자에게 권한 부여 가능
  → 비교적 열려 있음

제공 데이터:
  ✅ 노출 수, 클릭 수, 방문 수
  ✅ 키워드별 검색량 (월별 PC/모바일)
  ✅ 경쟁도 분석, 연관 키워드
  ✅ CTR, QC, Depth 정보
  ✅ 캠페인/광고그룹/소재 관리
  ✅ 보고서

2026 변화: 네이버 광고 플랫폼 통합 예정
  (검색광고 + 성과형 DA → 단일 도메인)
```

## 5. 구조적 불완전 요약

```
서비스의 핵심 가치: "전 채널 통합 AI 브리핑"

이것이 성립하려면:
  ✅ 네이버 데이터      → 접근 가능
  🔴 카카오 데이터      → 접근 불확실 (공식대행사 전용)
  🟡 메타 데이터        → 접근 가능 (앱 심사)
  🟡 구글 데이터        → 접근 가능 (토큰 승인)

결과:
  "통합"이라고 하면서 카카오가 빠지면?
  → 한국 소상공인 광고의 핵심 채널 누락
  → "통합" 타이틀의 신뢰도 하락
  → 경쟁력 반감
```
