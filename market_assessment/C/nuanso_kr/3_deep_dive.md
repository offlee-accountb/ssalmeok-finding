# Nuanso KR — Tier 3: 기술 설계, 수익 모델, 리스크 상세

> 최종 수정: 2026-02-19

---

## 1. 한국판 기능 설계

```
Nuanso KR v1.0 — "광고 브리핑 AI"
═══════════════════════════════════

[Core 1] 📊 광고 계정 통합 연결
├── 네이버 검색광고 API (✅ 접근 가능)
├── 네이버 성과형 디스플레이 광고 API (✅)
├── 카카오모먼트 API (🔴 공식대행사/사전협의 전용)
├── Meta Marketing API (🟡 앱 심사)
├── Google Ads API (🟡 토큰 승인)
└── (선택) 쿠팡 광고, 당근마켓 광고

[Core 2] 🤖 AI 주간 브리핑 (핵심 차별점)
├── 매주 월요일 아침 자동 발송
├── 한글 + 구어체 (전문용어 최소화)
├── 예시:
│   "지난주 네이버 검색광고 요약:
│    '봄 원피스' 키워드 클릭단가 23% 상승.
│    경쟁 상점 1주일새 4곳 증가.
│    → 추천: '봄 린넨 원피스'로 롱테일 전환.
│    예상 절감: 주당 ~15,000원"
│
├── 채널별 성과 비교:
│   "네이버 ROAS 320% vs 메타 ROAS 180%
│    → 이번 주 네이버 예산 5만원↑, 메타는 소재 교체"
│
└── 경쟁사 동향 (가능 범위):
    "같은 업종 평균 클릭률 2.3% → 내 클릭률 1.8%
     → 광고 소재 교체 추천"

[Core 3] 💡 원클릭 액션 제안
├── "예산 재배분하기" → 확인 후 적용
├── "이 키워드 중단하기" → 확인 후 적용
├── "새 키워드 추가하기" → 제안 목록에서 선택
└── (고급) AI가 소재 이미지/카피 초안 생성

[Core 4] 📱 알림 채널
├── 카카오톡 알림톡 (한국 최적)
├── 슬랙 연동 (소규모 팀)
├── 이메일 주간 리포트
└── 앱 푸시 (모바일 앱 버전)

[Core 5] 📈 리포트 자동 생성
├── 월간 광고 성과 리포트 PDF
├── 클라이언트 보고서 (대행사용)
└── 전년/전월 대비 시각화
```

## 2. 타겟 고객

| 세그먼트 | 규모 | 페인포인트 | 지불의사 |
|----------|------|-----------|---------|
| 스마트스토어 셀러 | 50만명+ | 네이버 광고 수치 해석 못 함 | 월 3~5만원 |
| 소상공인 (오프라인) | 350만명 | 카카오/네이버 광고 어려움 | 월 2~3만원 |
| 1인 D2C 브랜드 | 5만명+ | 멀티채널 운영 | 월 5~10만원 |
| 소규모 마케팅 대행사 | 1만곳+ | 클라이언트 다수 관리 | 월 10~30만원 |

## 4. 기술 스택 (MVP)

```
Frontend:   Next.js + React + Chart.js (시각화)
Backend:    Python FastAPI (데이터 분석 특화)
DB:         PostgreSQL + TimescaleDB (시계열)
AI:         GPT-4o-mini (브리핑 생성) + 자체 룰 엔진 (액션)
API 연동:
  ├── 네이버 검색광고 API (공식, ✅)
  ├── 카카오모먼트 API (공식, 🔴 접근 제한)
  ├── Meta Marketing API (공식, 🟡)
  └── Google Ads API (공식, 🟡)
알림:       카카오 알림톡 API
결제:       TossPayments
배포:       AWS
```

## 6. 실행 로드맵 (만약 진행 시)

```
Phase 1 (Month 1-2): MVP — 네이버 단채널
├── 네이버 검색광고 API 연동
├── AI 주간 브리핑 생성
├── 기본 대시보드
└── 베타 유저 50명 모집 (스마트스토어 셀러)

Phase 2 (Month 3-4): 유료화 + 메타 추가
├── Meta Marketing API 연동
├── 2채널 통합 브리핑
├── TossPayments 결제
└── 카카오톡 알림 연동

Phase 3 (Month 5-6): 카카오 접근 시도
├── 카카오 파트너십/대행사 등록 시도
├── Google Ads API 추가
├── 액션 제안 기능
└── 에이전시 플랜 런칭

Phase 4 (Month 7+): 스케일
├── 리포트 자동화
├── AI 소재 카피 생성
├── B2B 영업 (프랜차이즈 본사 등)
└── 대행사 시장 공략
```

## 7. C등급 → B등급 승격 조건

```
다음 중 하나가 해결되면 B로 승격 가능:

1. 카카오모먼트 API 접근 확보
   → 공식대행사 등록 or 카카오 파트너십 체결
   → 이 하나만 해결되면 구조적 완전성 확보

2. 카카오 없이도 가치 증명
   → "네이버+메타" 2채널만으로 유료 유저 500명+ 확보
   → "통합"의 정의를 "네이버+메타+구글"로 재설정
   → 카카오는 "추가 채널"로 포지셔닝

3. 카카오가 자체 API 정책 완화
   → 2026 하반기 카카오 광고 플랫폼 고도화 시 가능성
```
