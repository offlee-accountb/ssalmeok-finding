# Termsy KR — 약관 AI 분석 (C등급)

> 판정: C등급 — 구조적 접근성 최고, 수익화 불명확

## 한 줄 요약
Chrome 확장으로 약관/이용약관을 AI가 분석해서 "이 서비스 위험해요" 알려주는 도구

## 구조적 접근성: ✅✅✅ 최고
- 약관 = 100% 공개 정보 → 법적 문제 없음
- Chrome 확장에서 현재 페이지 텍스트 추출 → API 불필요
- LLM API만 있으면 구현 가능 → 기술 난이도 낮음

## C등급 사유: 수익화 불명확
- 개인: "약관 분석에 돈 낸다?" → 무료 기대
- B2B: 법무팀 있는 기업은 자체적으로 함
- 유저 액션 불명확: 약관이 나빠도 서비스 안 쓸 수 없음

## B등급 승격 조건
- B2B 피벗: 기업 법무/컴플라이언스 팀 대상 약관 모니터링 SaaS
- 또는: "내 개인정보가 얼마나 노출되어 있는지" 대시보드로 확장
