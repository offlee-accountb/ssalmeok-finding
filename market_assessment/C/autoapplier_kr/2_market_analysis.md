# AutoApplier KR — Tier 2: 시장 및 경쟁 분석

> 최종 수정: 2026-02-19

---

## 1. 원본 제품 상세

### AutoApplier (미국)

**핵심 기능 5가지:**

| # | 기능 | 설명 |
|---|------|------|
| 1 | AI Job Agent | 클라우드 브라우저가 채용사이트에서 실제 지원서 작성+제출 (하루 수백 건) |
| 2 | AI Interview Buddy | 폰 마이크로 면접 대화 실시간 청취 → 맞춤 답변 제안 |
| 3 | AI LinkedIn Bot | LinkedIn Easy Apply 자동화 (Chrome 확장) |
| 4 | AI Cover Letter | JD+이력서 → 맞춤 커버레터 (ATS 최적화) |
| 5 | AI Resume Generator | ATS 통과율 높이는 이력서 자동 생성 |

### 미국 경쟁사 현황

| 경쟁사 | 가격 | 차별점 |
|--------|------|--------|
| LazyApply | $99 일회성 or 구독 | LinkedIn/Indeed/ZipRecruiter 자동 지원 |
| JobCopilot | ~$56/월 | 하루 50건 자동 지원, 콜백률 2% 미만 |
| Scale.jobs | $199 일회성 | AI+사람 결합, 콜백률 47% |
| Sonara | 구독형 | 프로필 기반 자동 탐색+지원 |
| AIApply | $29/월+크레딧 | 크레딧 기반 건당 과금 |

### 핵심 발견: 자동 제출의 현실

> **전 세계 어떤 채용사이트도 "구직자 자동 지원" API를 제공하지 않음.**
> LinkedIn, Indeed 모두 약관에서 봇/자동화 명시적 금지.
> AutoApplier 등은 전부 **브라우저 자동화(봇)**으로 구현.
> → 계정 정지 리스크 있음.

## 2. 한국 시장 현황

### 기존 플레이어 분석

| 서비스 | 영역 | 뭘 하나 | 약점 |
|--------|------|---------|------|
| **사람인** | 공고 탐색 | 월 1,300만 트래픽. AI 자소서 코치 | 사람인 안에서만 작동 |
| **잡코리아** (워크스피어) | 공고 탐색 | LOOP AI 추천 (280% 지원 증가) | 잡코리아 안에서만 작동 |
| **원티드** | 공고+매칭 | OpenAPI 제공, AI 합격 예측 | 경력직 편향 |
| **자소설닷컴** | 자소서 참고 | 합격 자소서 수만 건 축적 | 읽기만 가능, AI 분석 없음 |
| **링커리어** | 정보+커뮤니티 | 대기업 채용 정보, 커뮤니티 | 자소서 생성 기능 없음 |
| **뤼튼** | AI 작성 | 무료 AI 자소서, 한국어 최적화 | 채용 특화 아님, 범용 도구 |
| **ChatGPT** | AI 작성 | 자소서 작성 보조 | 채용 특화 아님, 범용 도구 |
| **무하유 PRISM** | AI 탐지 | AI 자소서 탐지+면접 평가 | ⚠ 기업용 (구직자 적) |
| **캐치** | 공고+알림 | 대기업 공채 알림 | 자소서 기능 없음 |

### 한국에 있는 것 vs 없는 것

```
✅ 이미 있는 것 (레드오션):
├── 채용공고 탐색·추천
├── AI 자소서 초안 생성
├── 합격 자소서 참고
├── 공채 알림+캘린더
└── AI 면접 평가 (기업 측)

❌ 한국에 없는 것 (기회):
├── 크로스 플랫폼 지원 현황 통합 관리
├── 자소서 버전 관리 (기업별/수정 이력)
├── 합격 자소서 패턴 AI 분석 (구조적 인사이트)
├── AI 면접 실시간 코칭 (구직자 측)
└── 기업 자체 채용 홈페이지 정보 통합
```

## 4. 데이터 소스 가용성

### 공식 API (합법적 접근)

| 소스 | API | 제공 데이터 | 신청 |
|------|:---:|-----------|------|
| 사람인 | ✅ | 채용공고 검색 | 신청→승인 |
| 잡코리아 | ✅ | 채용정보 XML, 공채달력 | 신청→승인 (링크 연결 필수) |
| 원티드 | ✅ | 포지션, 기업, 직군/직무/스킬, 합격 예측 | 등록→인증키 (3영업일) |

### 대기업 자체 채용 홈페이지

| 기업 | URL | 로그인 필요 | 자소서 항목 공개 |
|------|-----|:---:|:---:|
| 삼성 | samsungcareers.com | ❌ | ✅ |
| 현대 | hyundai.com/careers | ❌ | ✅ |
| SK | skcareers.com | ❌ | ✅ |
| LG | careers.lg.com | ❌ | ✅ |

> **대기업 자소서 항목은 공개 정보 → 인간이 직접 수집+등록하는 방식이 가장 안전하고 저렴**  
> (시즌당 50곳 × 30분 = 25시간 ≈ 40만원)

### 법적 판례

> **잡코리아 vs 사람인 대법원 판결:**  
> 사람인이 잡코리아 공고를 무단 크롤링 → 패소 (2,000만원 배상)  
> "부당한 무임승차" 인정.  
> → 공식 API 사용 필수. 무단 크롤링 불가.

## 5. 경쟁 우위 분석

### 가능한 차별화 포인트

| 전략 | 가능성 | 이유 |
|------|:---:|------|
| "더 좋은 AI 자소서" | ❌ | 이미 commodity. 뤼튼/ChatGPT 무료 |
| "더 많은 공고" | ❌ | 사람인/잡코리아 독점 |
| "자동 제출" | ❌ | 한국 사이트 구조상 어렵고 법적 리스크 |
| **"여정 허브"** | 🟡 | 크로스 플랫폼 관리는 빈자리. 단, 검증 안 됨 |
| **"합격 패턴 분석"** | 🟡 | 데이터 축적 필요. 초기에는 공개 자소서 분석 |
| **"지원 그룹 커뮤니티"** | 🟡 | 네트워크 효과 가능. 초기 유저 모으기 어려움 |

### "여정 허브" 전략 상세

**핵심 아이디어:** 기존 서비스를 대체하지 않고, 연결하는 허브

```
사람인/잡코리아/원티드 = "마트" (공고 제공)
삼성/현대/SK/LG = "브랜드 직매장" (직접 채용)
자소설/뤼튼 = "재료상" (참고 자료, AI 도구)

우리 = "장바구니+가계부" (전체 관리)

마트를 만들 필요 없이, 마트 장보기를 관리해주는 앱.
```

**서비스 구성:**
- 전 플랫폼 공고 한 곳에서 탐색 (API 연동)
- 기업별 자소서 항목 + 필요서류 안내 (인간 큐레이션)
- 자소서 AI 생성 + 버전 관리
- 면접 코칭
- 지원 현황 통합 대시보드
- (데이터 축적 후) 합격 패턴 분석
