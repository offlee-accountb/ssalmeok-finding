## 한국 시장 적합성

| 항목 | 판정 | 이유 |
|------|:---:|------|
| 수요 존재? | ✅ | 구직자 100만+/년, 자소서 고통 |
| 경쟁 강도 | 🔴 | 사람인, 잡코리아, 자소설닷컴, 뤼튼 등 레드오션 |
| 시장 방향 | 🔴 | 대기업 공채 43% 감소, 수시 채용 전환 중 |
| 고객 지불의사 | 🔴 | 취준생은 가격 민감 (무료 대안 다수) |
| 기술 실현성 | ✅ | 공식 API 3곳(사람인/잡코리아/원티드) + LLM |
| 법적 리스크 | ✅ | API 사용 = 합법 (자동제출 안 함) |
| 차별화 가능성 | 🟡 | "여정 허브" 컨셉은 빈자리지만 검증 안 됨 |

## 최종 판정

```
종합 점수: ★★☆☆☆ (5점 만점 중 2점)

이유:
├── "더 좋은 AI 자소서"는 차별점이 안 됨 (commodity)
├── 시장 자체가 축소 중 (공채 → 수시)
├── 기존 강자(사람인, 잡코리아)의 헤게모니 강력
├── 고객(취준생)의 지불 여력 낮음
└── 규모의 경제 없이는 수익성 확보 어려움

결론: 단독 사업으로는 비추천.
      대형 플랫폼의 부가 기능 or 대규모 투자 기반으로만 가능.
```

## 유일한 가능성: "여정 허브" 전략

기존 서비스를 **대체**하지 않고 **연결**하는 허브.
크로스 플랫폼 지원 현황 추적 + 자소서 버전 관리 + 결과 분석.
→ 사람인에도 없고, 잡코리아에도 없는 빈자리.
→ 단, 시장 검증 필요.
