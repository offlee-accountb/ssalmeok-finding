## 1. 한국판 기능 설계

### 코어 기능 (MVP)

```
AutoApplier KR v1.0
═══════════════════

[Core 1] 🔍 공고 통합 수집 + 매칭
├── 사람인 API + 잡코리아 API + 원티드 API
├── 관심 직군/지역/연봉/기업규모 필터
├── AI 매칭 점수: "이 공고, 너한테 87% 맞아"
└── 마감일 알림

[Core 2] ✍️ 자소서 AI 생성 (한국 특화)
├── 기업별 인재상 자동 분석 → 반영
├── 항목별 생성: 성장과정 / 지원동기 / 입사 후 포부 / 장단점
├── NCS 역량 키워드 자동 삽입
├── 기업별 합격 자소서 패턴 학습
└── 글자수 자동 맞춤 (500자/1000자/1500자)

[Core 3] 🎤 AI 면접 코치
├── 인성면접 시뮬레이션
├── 직무면접 예상 질문 + 모범답안
├── 답변 분석: 길이, 구조, 키워드, STAR 기법
├── 음성 분석: 속도, 어/음 빈도 (STT 기반)
└── (고급) 실시간 면접 코칭

[Core 4] 📊 지원 대시보드 (여정 허브)
├── 전 플랫폼 지원 현황 통합 관리
├── 자소서 버전 관리 (기업별)
├── 서류통과/면접/결과 추적
├── 마감일 캘린더
└── 지원 통계 분석

[Core 5] 🏢 대기업 정보 (인간 큐레이션)
├── 대기업 30~50곳 자소서 항목+글자수
├── 필요 서류 체크리스트
├── 인재상 키워드
├── 채용 홈페이지 직접 링크
└── 시즌별 업데이트 (인간 작업)
```

## 2. 타겟 고객

| 세그먼트 | 규모 | 페인포인트 | 지불의사 |
|----------|------|-----------|---------|
| 대졸 신입 구직자 | ~50만명/년 | 50곳+ 지원, 자소서 노가다 | 월 2~5만원 |
| 이직 준비생 | ~30만명/년 | 현직 유지하면서 몰래 지원 | 월 3~5만원 |
| 경력직 전환자 | ~20만명/년 | 업종 전환, 이력서 재구성 | 월 5만원+ |
| 공시생 전환자 | ~10만명/년 | 민간 채용에 익숙하지 않음 | 월 2~3만원 |

## 4. 기술 스택 (MVP)

```
Frontend:   Next.js + React
Backend:    Python FastAPI
DB:         PostgreSQL + Redis
AI:         OpenAI API (GPT-4o-mini 자소서, Whisper STT)
크롤링:     없음 (공식 API 사용)
결제:       TossPayments / 아임포트
배포:       Vercel + AWS
알림:       카카오 알림톡 API (선택)
```

### API 연동 상세

| API | 용도 | 난이도 | 비용 |
|-----|------|:---:|------|
| 사람인 채용 API | 공고 수집 | 중 | 무료 (확인 필요) |
| 잡코리아 채용 XML | 공고 수집 | 중 | 무료 (링크 연결 필수) |
| 원티드 OpenAPI | 공고+기업+매칭 | 중 | 무료 (인증키 필요) |
| OpenAI API | 자소서 생성, STT | 낮 | ~$0.15/1M tokens |
| TossPayments | 결제 | 낮 | 3.3% 수수료 |

## 6. 리스크 매트릭스

| 리스크 | 수준 | 영향 | 대응 |
|--------|:---:|:---:|------|
| "AI 자소서"가 commodity | 🔴 높 | 치명적 | 자소서 생성만으로는 안 됨 → 여정 관리 차별화 |
| 사람인/잡코리아 자체 AI 강화 | 🔴 높 | 심각 | 크로스 플랫폼 포지셔닝으로 회피 |
| 무하유 PRISM (AI 자소서 탐지) | 🟡 중 | 보통 | "완전 AI 생성"이 아닌 "인간 편집 보조" 포지셔닝 |
| 공채 시장 축소 | 🔴 높 | 장기 | 수시 채용 대응 기능 추가 필요 |
| 취준생 지불 여력 부족 | 🔴 높 | 치명적 | 프리미엄 + B2B(대학 취업센터) 병행 |
| 시즈널 매출 편차 | 🟡 중 | 보통 | 이직 준비생(연중) + 시즌 패스로 보완 |
| 윤리 논란 (AI가 자소서 써도 되나) | 🟡 중 | 보통 | "보조 도구" 마케팅, 최종 편집은 유저 |

## 7. 실행 로드맵 (만약 진행 시)

```
Phase 1 (Month 1-2): MVP
├── 사람인+원티드 API 연동
├── 자소서 AI 생성 (기본)
├── 대기업 10곳 자소서 항목 수동 등록
└── 랜딩 페이지 + 베타 유저 모집

Phase 2 (Month 3-4): 유료화
├── 잡코리아 API 추가
├── 면접 코치 (텍스트)
├── 지원 대시보드
├── TossPayments 결제 연동
└── 대기업 30곳 확장

Phase 3 (Month 5-6): 차별화
├── 면접 음성 분석 (STT)
├── 자소서 버전 관리
├── 합격 패턴 분석 (데이터 축적 시)
└── 대학/기관 B2B 영업

Phase 4 (Month 7+): 스케일
├── 지원 그룹 커뮤니티
├── 수시 채용 대응
├── 대기업 50곳 확장
└── 마케팅 본격화
```

## 8. 최종 판정 상세

### 이 아이디어를 살리려면 반드시 필요한 조건:

```
1. 규모의 경제 (유저 최소 5,000+)
   → 마케팅 비용 필요 (월 500만원+)
   → 공채 시즌에 맞춘 바이럴 전략 필수

2. "여정 허브" 컨셉의 시장 검증
   → 유저가 실제로 "크로스 플랫폼 관리"를 원하는지 확인
   → MVP로 100명 테스트 필요

3. B2B 채널 확보
   → 대학 취업센터, 직업훈련원 단체 도입
   → 이쪽에서 안정적 기반 수익 확보

4. AI 자소서 탐지 우회 전략
   → 완전 자동 생성 ❌
   → "아웃라인 제안 + 유저 편집" 방식 ✅
```
