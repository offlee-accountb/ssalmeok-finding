## 📊 실제 매출이 확인된 로컬 AI 제품들

### 1. Rewind AI → Limitless AI (Meta 인수)
- **뭔데**: Mac에서 화면/음성을 24시간 녹화하고 AI로 검색 가능하게 만드는 앱
- **매출**: ARR $707K (2023.4), 밸류에이션 $350M (2023.5)
- **투자**: 총 $81M
- **결과**: 2025.12 Meta에 인수 → 원래 앱 종료
- **핵심 인사이트**: "로컬 화면 기억" 시장이 메타가 살 만큼 가치가 있다
- **한국 적용**: ✅ 높음 — 한국 기업 보안 환경에서 클라우드 불가, 로컬 필수

### 2. Screenpipe (Rewind 오픈소스 대체재)
- **URL**: https://screenpi.pe
- **뭔데**: Rewind 종료 후 등장한 오픈소스 화면/음성 녹화+AI 검색
- **특징**: Mac/Windows/Linux 지원, 완전 로컬, 플러그인 생태계
- **한국 적용**: ✅ — 한국어 OCR/STT 붙이면 "회의 내용 로컬 검색" 가능

### 3. Magnific
- **뭔데**: AI 이미지 업스케일링 도구
- **매출**: **$10M ARR**, 직원 **2명**
- **수익화**: 구독제
- **핵심 인사이트**: AI 도구에서 "2인 $10M"이 가능하다는 증거
- **한국 적용**: 🟡 (이미지 처리라 직접 관련은 낮지만 비즈니스 모델 참고)

### 4. Jan AI
- **URL**: https://jan.ai
- **뭔데**: 로컬에서 돌리는 ChatGPT 대안 (오픈소스)
- **트랙션**: **400만+ 다운로드**, Mac/Win/Linux
- **수익화**: 아직 오픈소스 중심(수익 모델 미공개)
- **한국 적용**: ✅ — "한국어 특화 로컬 ChatGPT" 포지셔닝 가능

### 5. LM Studio
- **뭔데**: 로컬 LLM 실행/관리 데스크톱 앱
- **수익화**: 무료 기본 + **Pro 구독** + **Enterprise 플랜** (SSO, 접근 제어)
- **핵심 인사이트**: "로컬 LLM 실행기"만으로도 엔터프라이즈 매출 가능
- **한국 적용**: ✅ — 한국 기업용 "사내 AI 실행 관리도구" 수요 확실

### 6. Ollama
- **투자**: YC 투자 ($125K)
- **수익화**: Pro $20/월, Max $100/월 (클라우드 기능), 로컬은 무료
- **전략**: 로컬 무료 → 클라우드 유료로 전환 유도
- **한국 적용**: ✅ — 이미 한국 개발자 생태계에서 널리 사용 중

### 7. Khoj AI
- **URL**: https://khoj.dev
- **뭔데**: 개인 AI 지식 관리 (NotebookLM 대안, 셀프호스팅 가능)
- **투자**: YC Seed $500K
- **상태**: 매출 발생 중 (금액 미공개)
- **한국 적용**: ✅ — "로컬 세컨드 브레인" = 정익님 Me.bot 아이디어와 일치

## 💡 1인 개발자가 할 만한 로컬 AI 제품 (필터링 후)

### ⭐ A급 (높은 가능성)

#### 1. 한국어 Screenpipe — "회의 기억 로컬 검색"
- **컨셉**: 화면/음성 로컬 녹화 → 한국어 OCR/STT로 검색
- **왜**: Rewind가 증명한 시장 ($350M), Screenpipe가 오픈소스로 깔아줌
- **차별화**: 한국어 STT (Whisper Korean fine-tune), 한국 기업 보안 규정 충족
- **수익화**: 개인 무료, 기업 월 ₩15,000~30,000
- **기술**: Tauri + Whisper + 로컬 임베딩
- **리스크**: 화면 녹화에 대한 심리적 저항, 스토리지 관리

#### 2. 로컬 문서 AI 비서 — "사내 문서 검색 + 요약"
- **컨셉**: 회사 문서(HWP/PDF/DOCX) → 로컬 임베딩 → AI 질의응답
- **왜**: Khoj/PrivateGPT가 증명, 한국은 HWP 때문에 외국 도구 직접 사용 불가
- **차별화**: HWP 파싱 내장, 한국어 RAG 최적화
- **수익화**: 1인 ₩9,900/월, 팀 ₩29,000/월
- **기술**: Tauri + Ollama + 벡터DB + HWP 파서
- **리스크**: HWP 파싱 품질, 대용량 문서 처리 속도

#### 3. AI 파일 정리앱 — "로컬 파일 자동 분류"
- **컨셉**: 바탕화면/다운로드 폴더 → AI가 내용 읽고 자동 정리
- **왜**: AI File Organizer Pro가 $12.99 일회성으로 팔리고 있음
- **차별화**: 한국어 파일명 인식, HWP/한글 문서 내용 분석
- **수익화**: ₩15,000 일회성 or ₩5,000/월
- **기술**: Tauri + Ollama + 파일 시스템 API
- **리스크**: 파일 잘못 이동하면 신뢰 붕괴 → 미리보기/언두 필수

### 🟡 B급 (가능하지만 경쟁 있음)

#### 4. 로컬 AI 글쓰기 보조 — "보고서/기안 작성 도우미"
- **컨셉**: 한국 공무원/기업 보고서 양식에 맞춘 로컬 AI 글쓰기
- **차별화**: 관공서 양식 사전 학습, 로컬이라 보안 규정 통과
- **리스크**: Claude/GPT 품질 대비 로컬 모델 품질 아직 부족

#### 5. 개인 지식 관리 세컨드 브레인
- **컨셉**: 메모/문서/웹클립 → 로컬 벡터 검색 + AI 연결
- **차별화**: Notion/Obsidian 연동, 한국어 임베딩
- **리스크**: 기존 도구(Obsidian+AI 플러그인) 대비 차별화 어려움
