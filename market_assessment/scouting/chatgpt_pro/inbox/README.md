# 📥 스카우팅 결과 인박스

> 이 폴더에 GPT / Gemini / Claude의 Deep Research 결과를 **그냥 넣어두세요.**  
> 파일명만 아래 규칙을 따르면 나머지는 정리할 때 자동 분류됩니다.

---

## 파일명 규칙

```
[AI이름]_[날짜]_[메모].md
```

### 예시:
```
gpt_0301_첫번째.md
gemini_0301.md
claude_0302_부동산쪽.md
gpt_0305_2차.md
```

- **AI이름**: `gpt`, `gemini`, `claude` 중 하나
- **날짜**: `MMDD` (월일, 4자리)
- **메모**: 선택사항. 본인이 알아볼 수 있는 한 줄 메모

> ⚠️ 파일 내용은 AI가 뱉어준 그대로 복붙하면 됩니다. 포맷 정리는 나중에 합니다.

---

## 정리 요청 방법

결과가 어느 정도 쌓이면 나(Antigravity)에게:

```
"인박스 정리해줘"
```

라고 말씀하시면 아래를 자동 수행합니다:

1. 날짜 기준으로 라운드 자동 묶기
2. 라운드별 교차 검증 테이블 생성 (3/3, 2/3, 1/3 분류)
3. `../findings.md`에 누적 결과 업데이트
4. `../../1_strategy.md`의 탈락 항목 업데이트 (필요 시)
